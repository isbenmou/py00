# Python Garden Exercises

A collection of beginner-friendly Python exercises designed to practice the fundamentals of the language through a garden-themed project.

Topics covered include:

* Functions
* Parameters & Arguments
* User Input & Output
* Variables
* Type Casting
* Arithmetic Operations
* Conditional Statements
* Loops
* Iteration
* Recursion
* String Methods
* Type Hints

These exercises focus on building strong programming fundamentals by solving practical problems and writing clean, readable Python code.
